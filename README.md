## Descrição do Projeto
O **EcoCommerce** é uma loja online ecológica desenvolvida com o objetivo de incentivar um estilo de vida mais sustentável por meio de uma experiência de navegação simples, moderna e intuitiva.

O projeto apresenta uma proposta de e-commerce com foco em produtos ecológicos, organizados em categorias como **roupas**, **casa** e **beleza**, valorizando a conscientização ambiental, a praticidade e o design responsivo.

## Objetivo
Desenvolver um site de e-commerce com identidade visual sustentável, utilizando tecnologias front-end para criar uma interface agradável, responsiva e funcional, permitindo ao usuário navegar entre páginas, visualizar produtos e acessar descrições detalhadas.

## Tecnologias Utilizadas
- HTML5
- CSS3
- Bootstrap 5
- Bootstrap Icons
- Font Awesome

## Funcionalidades
- Página inicial com apresentação da loja
- Menu de navegação responsivo
- Carrossel de banners com destaques da proposta sustentável
- Sessão de destaque com chamada para compra
- Exibição de produtos organizados por categorias
- Página de produtos
- Página de contatos
- Página de carrinho
- Páginas individuais de descrição dos produtos
- Footer com área de conta, informações, novidades e redes sociais
